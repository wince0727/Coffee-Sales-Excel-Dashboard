# Coffee Sales Dashboard (Excel)

## Project Overview
This project is an interactive Excel dashboard built to analyze coffee sales performance across time, customers, products, and countries.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Timelines

## Dashboard Features
- Monthly sales trend analysis
- Coffee type comparison (Arabica, Excelsa, Liberica, Robusta)
- Top 5 customers by sales
- Country-wise sales contribution
- Interactive filters for date, coffee type, and size

## Files Included
- Coffee Sales Dashboard.xlsx – Excel dashboard file
- dashboard.png – Dashboard preview

1. Download the Excel file.
2. Open in Microsoft Excel.
